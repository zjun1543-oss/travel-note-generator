## 📱 手机端操作步骤

```
1. 打开飞书App
2. 创建新的"多维表格"
3. 点击第一个单元格
4. 长按 → 粘贴
5. 表格自动生成！
```

---

## 💻 电脑端操作步骤

```
1. 打开飞书网页版/客户端
2. 创建新的"多维表格"
3. 点击第一个单元格
4. 按 Ctrl+V / Cmd+V 粘贴
5. 表格自动生成！
```

---

## ⚠️ 重要提示

粘贴后需要：

1. **设置字段类型**：
   - "日期"列 → 设置为"日期"类型
   - "费用/金额"列 → 设置为"数字"类型
   - "推荐指数/必去吗/时间段/类型/旅行性质" → 设置为"单选"类型

2. **添加单选选项**：
   - 时间段：上午、下午、晚上、全天
   - 类型：景点、美食、住宿、交通、购物、其他
   - 推荐指数：⭐、⭐⭐、⭐⭐⭐、⭐⭐⭐⭐、⭐⭐⭐⭐⭐
   - 必去吗：必去、推荐、可去可不去、不推荐
   - 旅行性质：独行、情侣、家庭、朋友

3. **删除示例数据**：
   - 保留表头（第一行）
   - 删除示例数据行
   - 开始填入你自己的数据
