# 旅行内容创作库 - 飞书多维表格设计

## 📋 表格结构

### 表格1：行程记录表（主表）
这是你最常填的表，旅行时每天往里加数据。

| 字段名 | 类型 | 选项 | 说明 |
|--------|------|------|------|
| 日期 | 日期 | - | 当天日期 |
| 城市 | 文本 | - | 所在城市 |
| 时间段 | 单选 | 上午/下午/晚上/全天 | - |
| 类型 | 单选 | 景点/美食/住宿/交通/购物/其他 | - |
| 名称 | 文本 | - | 景点/餐厅等名称 |
| 地址 | 文本 | - | 详细地址 |
| 费用 | 数字 | - | 消费金额 |
| 停留时长 | 文本 | - | 如"2小时" |
| 推荐指数 | 单选 | ⭐/⭐⭐/⭐⭐⭐/⭐⭐⭐⭐/⭐⭐⭐⭐⭐ | - |
| 必去吗 | 单选 | 必去/推荐/可去可不去/不推荐 | - |
| 预约信息 | 文本 | - | 如"需提前预约" |
| 拍照Tips | 文本 | - | 拍照建议 |
| 避坑提醒 | 文本 | - | 注意事项 |
| 我的感受 | 文本 | - | 真实感受 |
| 照片 | 附件 | - | 上传照片 |

---

### 表格2：费用汇总表

| 字段名 | 类型 | 选项 | 说明 |
|--------|------|------|------|
| 类别 | 单选 | 交通/住宿/餐饮/门票/购物/其他 | - |
| 项目 | 文本 | - | 具体项目 |
| 金额 | 数字 | - | 费用金额 |
| 备注 | 文本 | - | 补充说明 |
| 日期 | 日期 | - | 消费日期 |

---

### 表格3：旅行信息表

| 字段名 | 类型 | 选项 | 说明 |
|--------|------|------|------|
| 旅行名称 | 文本 | - | 如"韩国釜山5日游" |
| 目的地 | 文本 | - | 主要目的地 |
| 开始日期 | 日期 | - | 旅行开始日 |
| 结束日期 | 日期 | - | 旅行结束日 |
| 天数 | 数字 | - | 旅行天数 |
| 总预算 | 数字 | - | 预算金额 |
| 实际花费 | 数字 | - | 实际总花费 |
| 出发地 | 文本 | - | 从哪里出发 |
| 旅行人数 | 数字 | - | 几个人 |
| 旅行性质 | 单选 | 独行/情侣/家庭/朋友 | - |

---

## 🚀 在飞书中创建步骤

1. 打开飞书App或网页版
2. 点击"新建" → "多维表格"
3. 命名为"旅行内容创作库"
4. 创建3个表格：
   - 表格1：行程记录
   - 表格2：费用汇总
   - 表格3：旅行信息
5. 按上面的字段设计添加列
